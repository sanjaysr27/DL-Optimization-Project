# NVIDIA Jetson Orin Nano Configuration

## Hardware
- GPU: NVIDIA Ampere Architecture
- CUDA Cores: 1024
- Tensor Cores: 32
- GPU Performance: 1.33 TFLOPS
- CPU:
  - 6-core ARM Cortex-A78AE v8.2 64-bit CPU
  - 1.5 MB L2 Cache + 4 MB L3 Cache

## Memory and Storage
- RAM: 8 GB LPDDR5
- Memory Bandwidth: 102 GB/s
- Storage:
  - microSD support
  - External NVMe support

## Software Environment
- TensorFlow: 2.6.2

## Purpose
Used for edge inference benchmarking:
- Optimized model deployment
- Latency evaluation
- Memory footprint analysis
- Comparison of compressed models
